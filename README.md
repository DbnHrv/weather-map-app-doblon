# 🌦 Weather Map App

## 📌 Description
The Weather Map App is a simple and responsive web application that allows users to search for any city and view its current weather information along with its location displayed on an interactive map. It integrates real-time weather data and mapping services to provide an informative and user-friendly experience.

## ✨ Features
- 🔍 Search weather by city name  
- 🌡 Displays temperature in Celsius  
- ☁ Shows weather condition/description  
- 🗺 Interactive map with location marker  
- 📱 Responsive design (works on mobile and desktop)  
- ⚠ Error handling for invalid input and API issues  

## 🔌 API Used
- **OpenWeather API** – for fetching real-time weather data  
- **Google Maps JavaScript API** – for displaying map and location marker
